Staff Dashboard

  • Questionnaire

    Go to Create Questionnaire

    Fields (most are mandatory)
    Title
    Short but descriptive title for all users
    Active
    Will show up for Subjects if checked (hidden if not)
    Description
    Description for all admin users only - will not be displayed to Subjects
    Introduction
    First page of questionnaire to explain to Subjects what it is about and what to do - full WYSIWYG editor
    Code
    A reference code for the questionnaire - will show in lists
    Type
    Choose whether every question is on a separate page (multi-page) or all on one page (single page)
    Category
    Select one or more visit categories
    Group
    Select one or more groups to ensure questionnaire is only given to the correct group - leave blank for all

  • Questions

    Scroll to the bottom of the questionnaire page (questions only - answers are separately added). Note 3 questions are shown initially, if not required just use two and save. If more required, just click on the "Add another question" at the bottom.

    Fields
    Question Text
    Short textual question (max of 200 characters)
    Question Image
    An option to have an image instead with the question text showing as alternative and as popup text. Best results when the image is resized prior to upload - displayed at 400px width max
    Sequence Order
    Mandatory field - allows manual ordering of questions. Note there is a batch process for this after the questionnaire is saved.
    Group
    Select one or more groups to ensure question is only shown to the correct groups - leave blank for all
    Type
    Select from Radio (single), Checkbox (many), Text field (free text) or Dropdown (long lists) - this can be changed at any time as it is independent of the choices
    Required
    Indicates whether the Subject must answer this question (ie optional text questions should uncheck this)
    Question Help
    Not currently implemented

  • Save and Check

    1. SAVE the questionnaire with one of the save options
    2. Ensure you have set the order for the questions:
        Go to the Questionnaire List
      1. check one or more questionnaires
      2. from the "Action" dropdown, select "Generate order sequences for questions"
      3. click on the "Go" button

  • Choices

      There are two ways to create Choices for the questions, first go to Question List
    1. Batch process: check one or more questions, then select the "Action" dropdown and select from preset lists. This will generate a set of choices for each question selected - overwriting any existing ones.
    2. Manually create the choices by selecting the question from the Question List then adding the Choices:
      Fields
      Choice Image
      Option to upload an image (preferably sized at 100px width or less)
      Choice Text
      Text to display
      Value
      Number (preferable) or word to indicate which choice was selected - this should have some relevance for analysing the results
      Show Label
      Option to show or hide the choice text (this can be applied as alternative to the image if not displayed)
      Group
      Select one or more groups to ensure choice is only shown to the correct groups - leave blank for all
    3. SAVE with one of the save options at the bottom of the page

  • Subject Visit

    Go to Create Subject Visit

    Fields (most are mandatory)
    Date visit
    Set to the date and time of the visit (in advance is fine)
    Subject
    Add a new Subject with the green "plus" button (if they are not in the list). This popups up a separate screen - edit username and password (remember this!). Groups will need to be added separately - see below. (Note, if the Subject has already been added to the Users, it will appear in the dropdown list.)
    Category
    Select Wave for this visit. (Note this is not editable)
    XNAT ID (optional)
    ID to link to Subject's record in XNAT
    Parent (optional)
    Select from list if already added to Users or click on plus as above

  • Subject Details

    Go to Add or Edit User This is required to add Groups to a Subject/Parent. Subjects (and parents) can be added here first and then Subject visits set up.

    Fields (most are mandatory)
    Username
    Edit username (no spaces)
    Password
    To change, click on the text "this form" in tiny font under the password field.
    Personal Information
    First name is used in the Welcome message. Email address is used for resetting a forgotten password but it must be unique so use only if you are allowing Subjects to change their own password.
    Active
    Will only be able to login if this is checked
    Staff status
    Staff only - most likely yourself
    Superuser status
    Highly restricted - only users who understand the system
    Groups Very Important
    Select one or more groups for the Subject - these should match with Questionnaire groups
    Other fields
    That's it - except you may need to enter a value in the "Last Login" - today is fine

    And don't forget to SAVE!
